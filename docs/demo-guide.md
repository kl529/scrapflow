### 색상 팔레트
```css
/* Primary Colors */
--primary-blue: #667eea
--primary-purple: #764ba2

/* Status Colors */
--success-green: #10b981
--warning-yellow: #f59e0b

/* Neutral Colors */

```

### 로고 사용법
- 최소 크기: 24x24px
- 여백: 로고 높이의 50% 이상
- 배경색이 어두울 때는 흰색 버전 사용
- 브랜드 색상과 조화롭게 배치

### 타이포그래피
- 제목: 볼드, 큰 크기
- 본문: 레귤러, 적당한 크기  
- 강조: 색상 또는 볼드 처리
- 가독성을 위한 충분한 행간
