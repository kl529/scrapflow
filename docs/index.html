북마크는 너무 많고, 스크린샷은 찍어뒀는데 나중에 뭘 의미하는지 모르겠어요

"유용한 내용인데 웹페이지가 사라져버렸어"

링크는 깨지고, 콘텐츠는 삭제되고... 디지털 정보의 휘발성이 너무 큰 문제예요

😮‍💨

"정보 정리가 너무 복잡해요"

캡처 → 저장 → 분류 → 태그 → 메모... 과정이 복잡해서 결국 안 하게 돼요

🤔

"이미지 속 텍스트를 검색하고 싶은데..."

스크린샷은 많은데 내용으로 검색이 안 돼서 찾기가 어려워요

✨ ScrapFlow로 해결하세요!

복잡했던 정보 관리가 이렇게 간단해집니다

⚡

3초 완료

Ctrl+Shift+S → 드래그 → 코멘트 → 저장
더 이상 복잡한 과정은 없습니다

OCR로 이미지 속 텍스트까지 추출
코멘트와 함께 통합 검색 가능

🔒

완전 오프라인

모든 데이터 로컬 저장
개인정보 걱정 없이 안전하게

🌟 주요 기능

📸

초고속 캡처

글로벌 단축키로 언제든지 즉시 스크린샷. 백그라운드에서 항상 대기 중입니다.

🔤

OCR 텍스트 인식

한국어/영어 텍스트 자동 추출. 이미지 속 내용도 검색으로 쉽게 찾을 수 있어요.

🎨

SNS 공유 이미지

예쁜 템플릿으로 자동 생성. Instagram, Twitter 등에 바로 공유 가능합니다.

🗂️

카테고리 관리

개발, 디자인, 비즈니스 등 색상별로 체계적 분류. 나만의 분류 체계를 만드세요.

🔍

통합 검색

코멘트와 OCR 텍스트를 동시 검색. 실시간으로 원하는 스크랩을 찾아보세요.

📊

활동 시각화

GitHub 스타일 히트맵으로 스크랩 활동 추적. 지식 축적 과정을 한눈에!
